Why to use axTable angular component?
For end-users great user experience easy to use with many features and customizations.
For developers is angular native it is built with angular directives. is very simple to implement and high customizable you can use html templates or javascript templates generation. records and columns virtual scroll which can handle easy 100k records and up to 1000 columns. no need to have advanced knowledge of angular or javascript writing necessary javascript code is very simple and intuitive.
DevTools console will crash when loading 100k records, so keep it closed.
To see real axTable performance keep DevTools console closed and disable Batarang Chrome extension. Use Timings button to see loading timings.
Select a profile from profiles list in Settings popup to see different use cases.
