DevTools console will crash when loading 100k records, so keep it closed.
Unigue features (which no other Javascript has)
- Configuration Profiles Select a profile from profiles list in Grid Settings and then press apply.
- Advanced setup for grouping properties Select a profile with grouping. Then click on edit icon for a group in Grouping by columns list, to see what you can setup as end-user.
- Filter by grouping value Select a profile with grouping. Then click on filter icon on any group header to open UI for filter by grouping values.
