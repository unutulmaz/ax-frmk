				Attributes:
                

For Toggle button
                tabindex if is set to -1 will disable access by keyboard navigation. Other value will enable this feature.
				btn-class class list for toggle button.
                btn-text javascript expression which text for.
                btn-html javascript expression which contains html to use as innerHTML for toggle button.
                btn-bind-html javascript expression for set bind-html-compile attribute for toggle button.
                show-clear-button for show a clear button, if control is ax-dropdown-list or ax-dropdown-table.
                caret-class css class for caret. By default is 'caret'. If caret-class = 'fa' , no caret will be created.
                caret-style style attribute for caret element.
                

For Popup
                ctrl reference to an javascript object to control popup. This will be accessed in popup with name 'launcher'.
                open-params references to javascript object list separated by ';'. These objects are accessed in onOpen config method as arguments. First argument is $event by default.
                template-url javascript expression for template url. If not set this attribute, popup template will be set by ax-dropdown-popup innerHTML.
                popup-class css class for popup element created.
                popup-width style width property for popup element created.
                popup-height style height property for popup element created.
                popup-max-height style max-height property for popup element created, otherwise is set to auto.
                close-on-blur for close popup on document click. t's not working if outside elements clicked invoke stopPropagation() for click event.
                close-on-mouseleave for close popup when mouse leave popup (must first be entered).
                close-on-destroy=false to disable close popup on element destroy.
                disable-animation to disable open/close animation effect.
                popup-relative-top css popup element top property relative to toggle button. By default is zero.
                popup-relative-left css popup element left property relative to toggle button. By default is zero.
                popup-absolute-top css popup element top property.
                popup-absolute-bottom css popup element bottom property.
                popup-absolute-left css popup element left property.
                popup-absolute-right css popup element right property.
                

Popup controller methods
                onOpen method is invoked before open popup has as arguments $event and params defined in open-params attribute. For continue open event execution must set this.openFinish = true.
                onOpenFinish method is invoked before make visible popup.
                onClose method is invoked on popup close event.
                close method to trigger popup close event.
