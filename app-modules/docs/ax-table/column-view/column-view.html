Column view template
Built-in view template 

				Attributes:
				bind-to the property name (field) of row object (dataItem - as scope variable), if is not the same as ax-column parent element. Required if you want to use a built-in type view template.
				view-type can be one of values text, number, date, datetime, boolean-checkbox, boolean-radio, gutter-icons, crud-buttons, custom. This will create a ax-column-view template with type = view-type.
				double-binding can be true for built-in view template, if column could be changed by user editing in other column.In usual case, one time binding is used for column view template.
				style the css style for ax-column-view template. Here you can use padding-left:0 to eliminate the default padding-left, or overflow-y=hidden if you have same problem with display elements, and the default overflow-y=scroll is not suitable for the case.
				date-format date-format if view-type = date
				datetime-format datetime-format if view-type = datetime
				locale locale format for view-type = number
				decimals decimals number for view-type = number
				use-grouping if use thousand separator for view-type = number
				minimum-fraction-digits minimum fraction digits for view-type = number
				maximum-fraction-digits maximum fraction digits for view-type = number
				format-style style attribute for toLocaleString option
				

Custom view template:
                

				

				For custom view-type, the current row object is named dataItem, which can be used to create your custom view. You can access the
				ax-table parent scope with $ctrl.$parent ($ctrl.$parent.myVariable).
				
For ax-table performance in custom ax-column-view, use one time binding::, if expression will not be changed by user while is viewed.
				
For custom view, you cannot sort, or filter data. In this case a better approach is to use axDataAdapter to create a new column, which will be sortable and filterable.
				
Custom view can be used for view nested properties, custom actions or anything else. This accept any html you need.
