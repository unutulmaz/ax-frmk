Column resize & auto fit width column
Column resized
Columns can be resized by user with drag the right margin of column header.
Using tds-resizable = true ax-table attribute you can enable column resize with drag from data cell right margin.
Column width auto fit to content
Auto fit feature work in two manners: if filtered records < 300 , calculate columns width for all records, else just for visible records.
Attributes:
- columns-autofit-enabled enable end user access to auto fit feature. It is disabled and readonly for paginate=false
- columns-autofit boolean - if true, ax-table will execute columnsAutoFitAll controller method on dataLoad event
You can use min-width style attribute of ax-column-view, when you need to have a minimum width for a column, after applying column auto-fit

End user can use this feature from Settings on click to icon and for individual column from column menu
