Built-in filter menu template 

				You can use filter-type for ax-column attribute or ax-column-menu-filter template with data-type attribute, both definitions are identical.
				
Attributes of ax-column-menu-filter template
				initial-filter-option specify which filter menu option will be shown initial.
				data-type can have one of the values string, date, datetime, boolean, number.
				bind-to if not present, ax-column bind-to attribute will be used
				invariant-field if column has an invariant field
				

Default options for built-in filter menus
				
Options are readed from $axTableConfig().filters object (/components/ax-components-config.js file) and are overwriteables.
