Column editor template
When you need to define special control for editor other then then ax-column-edit template, you'll define ax-column-editor.
1. With ax-dynamic-template-url 

        2. or with html content
        

        
Attributes:
        width attribute to overwrite column width, which is default width for editor control.
        colspan attribute for columns table, which is created with ax-form control. See Record editing for more detials.
        editor-column-index column-index (1 based) where to add control in editor, when editor has more than one column for arrange controls (columns-no attribute of ax-grid-editor).
