Built-in filter template 

				Attributes
				bind-toif not present, ax-column sortable attribute take precedence from bind-to ax-column attribute
				show-configif built-in filter has config button, this will be shown. By default config button is not shown.
				filter-typecan have one of the values: text, number, boolean, date, datetime, dropdown-list, dropdown-list-distinctvalues, custom
				display-nameis used when is an option in ax-column-filter-menu - the name which appear in UI for end-user in Column filter menu
				show-in-popup true if you want to show filter input controls in a popup. Additional you can set popup-width and popup-height attributes.
				

Filter with filter-type = dropdown-list-distinctvalues has in addition the following attributes:
				selectable-rows can be multiple,single.
				

Filter with filter-type = dropdown-list has in addition the following attributes:
				ctrlpopup controller object if you want to control behaviour of popup.
				datasourcedata collection (pointed to $ctrl.$parent from view controller scope). Is required.
				selectable-rowssingle/multiple. Is required.
				model-typecan have one of the values: object, id-field. Default it is id-field.
				item-id-fieldidentity field of datasource. Is required if model-type=id-field.
				item-display-fieldproperty name to display text, if it's different from id-field.
				item-invariant-fieldproperty name for invariant field, if column has.
				filter-strict="false"filter will be made for value included in item.property value.
				ng-modelif you want to overwrite bind-to attribute.
				ng-changeif you want to overwrite default ng-change handler event.
				ng-disabledif you want to disable control.
				disable-animationfor disable open/closing animation.
				close-on-blurfor closing popup on popup blur. By default is true.
				empty-option-texta javascript expression. By default is empty text.
				add-empty-optionif you want to add an empty option on top list options.
				show-searchif you want to have a text filter for datalist.
				show-clear-buttonif you want to show a clear button in filter dropdown when is something selected.
				item-show-check=falseif you want to disable show checkboxes for items. By default are showed.
				show-check-allif you want to show a button for check all options in a multiple selectable datalist.
				show-uncheck-allif you want to show a button for uncheck all options in a multiple selectable datalist.
				selectable-rows-checked-classcss class for checked item. By default is fa fa-check-square-o.
				selectable-rows-unchecked-classcss class for unchecked item. By default is fa fa-square-o.
				list-heightdatalist css height property. by default is 150px.
				list-widthdatalist css width property, if you want to have different width as column width. By default datalist has same width with column.
				popup-max-widthpopup max-width css property.
				popup-max-heightpopup max-height css property.
				
 For more details view page: ax-dropdown-list
