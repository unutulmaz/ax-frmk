Records ordering
Developer
- ax-table attribute order-by = "Column1,-Column2,Column3" columns list, separated by comma and using minus sign for descending order.
- ax-column attribute sortable = "ColumnNameInvariant" or empty. If empty the sorting will be made for column name defined in bind-to attribute. ColumnNameInvariant is a hidden column with value for ColumnName without accents. Can be created with axDataAdapter, open Javascript tab to see how.
- ax-column-header attribute sortable = "ColumnNameInvariant" or empty. If empty the sorting will be made for column name defined in bind-to attribute.

End user - UI
- each column has a menu activated by click on the last level header.
- on axTable Settings button open a config popup where you can set sorting by drag & drop. Here can be set order by multiple columns.
In this sample data are filtered with a custom filter by deliveryCity which has accents or non-alphabetic characters in name.
