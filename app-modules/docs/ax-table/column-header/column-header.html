                It's using header ax-column attribute as innerHTML

                If you need custom template for header, you need to use ax-column-header element.
                
Attributes
                row-index when you have a header with more than one row. If you use rowspan or/and colspan you must skip ax-column-header element for the those columns which are covered by rowspan and colspan
                header-title is necessary only when ax-column-header's innerHTML is not static html (text), but is generate dynamic by a function. Is used as label on Settings Columns Layout tree, and in editor (if ax-table attribute edit-row = editor).
                header-menu = 'false' for disables insert header menu in th cell. Default header menu is inserted on first th element without colspan.
                sortable if if want to define sortable in th cell, otherwise will be inserted to first th element without colspan.
                colspan how many columns will cover this header.
                rowspan how many rows will be expanded this header.
                title title attribute for header cell.
                style css style for header cell.
                
Rowspan and colspan work in same way as table th attributes. If you did some mistakes using colspan or rowspan, you'll receive an error message in DevTools console.
                
innerHTML of ax-column-header can be anything you need. On header ax-table parent scope will be accessed by $ctrl.$parent.

        No header
        
If you don't want to show column headers, you can use no-header="true" attribute for ax-table. This attribute is effective only if you don't put custom header template or filter template.
        
No pagination
        
Also is possible to hide pagination bar with paginator-show="false" attribute for ax-table.
