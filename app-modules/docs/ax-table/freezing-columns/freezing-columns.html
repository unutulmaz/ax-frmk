Freezing and hidding columns
Freezing using ax-table attributes
- freeze-columns-enabled - boolean - enable/disable freezing columns
- customizable-freezed-columns - boolean - enable/disable end user UI for customize freezed columns
- left-freezed-columns - integer>0
- right-freezed-columns - integer>0

End user UI:
- in Settings button, click on icon open config popup for customize left and right freezed columns.
- in column menu click on click on icon to set last left or right freezed column.
Hidden column on each column hidden-column an ax-column attribute. Use this attribute on columns you want to hides (without value). 

		End user UI:
		
 - in Settings button, click on icon  open config popup for hide/show columns. Columns hidden by hide-columns group option, cannot be made visible in this popup. Use group edit popup instead
		
 - in column menu click on click on icon  to hides column.
		

Developer can disable column access conditionally for user using forbiddenColumns controller method.
