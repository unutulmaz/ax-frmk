Exporting data
At this moment you can export to excel or as html page (ready to print). Data export is defined with an ax-export element as ax-table child.By default export is enabled. To disable export must user disable-export=true ax-data-tyable attribute. 

        Export template has attributes:
        file-name a name for generated file and report title.
        export-type with values: [client, server].
        api-controller if export-type = server the api controller for backend. This controller must have actions xlsExport - for xls export, and htmlExport for print export. For .NetCore you can use axTableController.cs
        data-value if export-type = client, how to fill cells. values are [bind-to,view].
        view-template-limit integer for switch to bind-to fill cells type, if export-type = client.
        

Access for user to export is made by pressing upload  button in Settings button
