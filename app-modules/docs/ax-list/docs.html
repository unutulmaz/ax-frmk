axList
axList derives from axTable with less features (no UI configuration, no pivot table mode, no javascript template generation). 

				axList specific attributes:
                no-header=false if you want to show columns headers. By default is true.
                paginator-show=true if you want to show paginator on bottom. BY default is false.
                datasource data collection (pointed to $ctrl.$parent from view controller scope). Is required.
                selectable-rows can by single or multiple. By default is single.
                selectable-rows-model reference to an scope variable where to store selected items. Object is type of array if selectable-rows='multiple'
                selectable-rows-model-type can have one of the values: object, id-field. Default it is id-field.
                select-on-click-row if you want to select item on click row event.
                item-id-field identity field of datasource. Is required.
                item-display-field property name to display text, if it's different from id-field.
                item-invariant-field property name for invariant field, if column has.
                on-selection-change a handler for selection change event.
                add-empty-option if you want to add an empty option on top list options.
                show-close-button if you want to show a popup close button when datalist is opened in a ax-dropdown-popup.
                
Attributes available if not columns template provided for datalist:
                show-search if you want to have a text filter for datalist.
                item-show-check=false if you want to disable show checkboxes for items. By default are showed.
                show-check-all if you want to show a button for check all options in a multiple selectable datalist.
                show-uncheck-all if you want to show a button for uncheck all options in a multiple selectable datalist.
                selectable-rows-checked-class css class for checked item. By default is fa fa-check-square-o for multiple selection mode and fa fa-check-circle-o for single section mode.
                selectable-rows-unchecked-class css class for unchecked item. By default is fa fa-square-o for multiple selection mode and fa fa-circle-o for single section mode.
                

Readonly attributes:
                apply-changes-on-save true.
                customizable-pivot-table false.
