Unigue features (which no other Javascript DataGrid has)
1. Configuration Profiles Select a profile from profiles list in grid Settings icon: and then press apply.
A profile can contains which columns are hidden, left and rigth columns freezing states, data ordering, data grouping definitions, and toolbar buttons
Profiles can be defined by developer and are readonly. This can be extende to end-user, to be able to save his own profiles and stored in backend database.
Click maximize icon from rigt upper corner for extend DataGrid to all workspace;
2. Fixed group header on horizontal scroll
3. Advanced setup for grouping properties: click on edit icon for a group in Grouping by columns list (Data Grouping section), to see what you can setup as end-user.
4. Filter by grouping value: click on filter icon on any group header to open UI for filter by grouping values.
